Items for coding python by vim.

# Usage
## python.vim
Download 'python.vim' and execute commands;

    $ mkdir ~/.vim/syntax
    $ mv python.vim ~/.vim/syntax/

Write in your '.vimrc' as follow;

    syntax on

## vimrc
Download 'vimrc' and execute commands;

    $ mv vimrc .vimrc
    $ mv .vimrc ~/
